# 热门新闻监控 (Hot News Monitor)

实时监控 AI 模型、编程动态和中山珠海地区 AI 岗位招聘的网页应用。

## 功能特性

- 🔥 监控 2 个新闻分类：
  - AI/模型/编程：AI 模型发布、编程技术动态
  - 中山珠海AI岗位招聘：中山珠海地区的 AI 相关职位
- ⏱️ 每 5 分钟自动刷新
- 🎨 现代简洁的 UI 设计
- 📊 实时显示统计数据

## 运行方法

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 启动应用：
```bash
python app.py
```

3. 打开浏览器访问：
```
http://localhost:5000
```

## API 接口

| 接口 | 说明 |
|------|------|
| `GET /` | 主页面 |
| `GET /api/news` | 获取所有新闻 |
| `GET /api/news/ai` | 获取 AI/模型/编程新闻 |
| `GET /api/news/jobs` | 获取招聘信息 |
| `GET /api/status` | 获取系统状态 |

## 新闻数据格式

每个新闻条目包含：
- `title`: 标题
- `source`: 来源
- `time`: 发布时间
- `category`: 分类标签

## 扩展说明

当前使用模拟数据。如需接入真实数据源，可以：
1. 接入新闻 API（如新浪新闻、腾讯新闻等）
2. 使用爬虫抓取目标网站
3. 接入招聘平台 API（智联招聘、前程无忧等）
